# 7evencord-SRC

A Decompiled Version of the controversial custom Discord client called **7evencord**
The Client sends your Discord Token **in Plain-Text** to the **Proxy** using an **Free Hosting Service**
There are more apps made by chockingnetdude with **Vulnerabilities** like **8Store**. 

## The Problem

**The issue with 7evencord is its handling of the Discord User Token. It had been found out that the client sends the Discord Token directly to a proxy in Plain-Text. This poses the high risk of your account being compromised! Any Developer should make sure that anyone can't access it publicly nor store them in plain-text.**
